# About the contents for this seminar
The contents are thought for people with little to none experience with Python, who might benefit from an introductory seminar focused on data analysis for radiation physics.
- The seminar follows the jupiter notebook `Basics-Data-Analysis_python_seminar.ipynb`
- The solution of the excercises can be found in the folder `solution/`
- The data used by the python code are created launching `_build/create_mock-datafiles.py`
- Output data-files from MAESTRO (.Spe) were indeed created manually to mock realistic measurements.
- `requirements.txt` lists the python's packages used by the jupiter notebook
- `mybinder_link.txt` contains the link to build a virtual python-jupiter environment using _`https://mybinder.org/`_. This allows the user to open the jupiter notebook and run it.
- `.dockerignore` was created to exclude some of the repo contents from the virtual environment created using _mybinder.org_
